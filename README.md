# 💬 Quick Chat

**Quick Chat** is a lightweight, real-time chat application built with modern web technologies. It offers a simple and responsive interface for users to communicate instantly, making it ideal for small team chats, project collaboration, or demo purposes.

---

## 🚀 Features

- 🔄 Real-time messaging
- 🧑‍🤝‍🧑 Multi-user support
- 💬 Live message updates using WebSockets or API polling
- 📱 Responsive UI for desktop and mobile
- ⚡ Fast and lightweight frontend

---

## 🛠 Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js (optional)
- **Communication:** Socket.io / REST APIs (based on your implementation)
